# BakeryPR

Bakery Processes Automation

Automate bakery processes from raw material to production and finally sales. It automatically calculates cost price for the production of each product and also manages reports and sales.

## Features

- Inventory management for ingredients and products
- Recipe management with ingredient tracking
- Production planning and cost calculation
- Sales tracking and revenue reporting
- User management with roles
- Overhead cost allocation
- License validation

## Technologies

- C# / WPF (Windows Presentation Foundation)
- .NET Framework
- SQLite (assumed from DAO structure)

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/olabodeaguda/bakery.git
   ```

2. Open `BakeryPR.sln` in Visual Studio.

3. Restore NuGet packages.

4. Build and run the application.

## Usage

Launch the application and log in with appropriate credentials. Navigate through the various modules to manage inventory, recipes, production, and sales.
